# TGA Truck Gear Autoparts

Premium Shacman truck spares website for TGA Truck Gear Autoparts.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy On Vercel

Use the GitHub repo:

```text
RonanJaeger721/TGA-
```

Recommended Vercel import settings:

```text
Project Name: TGA-
Framework Preset: Next.js
Root Directory: ./
Install Command: npm install
Build Command: npm run build
Output Directory: leave empty
Node.js Version: 22.x
```

No environment variables are required for this first version.

After importing the repo, Vercel will deploy from the `main` branch. Future
pushes to `main` will update the live site automatically.

## Useful Commands

```bash
npm run dev
npm run build
npm run start
```
